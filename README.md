PROJECT DESCRIPTION.
This is a system where a tenant can see vacant rooms and register for the one they desire,
pay rent from their acount, request for room maintainance and see changes done on the room by the management.
The landlord can see the occupied rooms and monitor payments for each tenants and also can manage the tenant requests in 
coordination with the caretaker.
